Installation (unter ubuntu)

  1. hier herunterladen, neuste Version des 'glgames' *.deb Archivs wählen
  2. das Archiv installieren (Doppelklick)

Installation (unter fedora)

  1. in einer Root-Konsole ausführen: 
    yum groupinstall "Development Tools"
yum install ruby rubygems ruby-devel

yum install SDL-devel SDL_ttf-devel SDL_image-devel freeglut-devel
gem install rubysdl ruby-opengl

yum install bitstream-vera-fonts
  2. hier herunterladen, neuste Version von 'glgames' wählen
  3. das Archiv entpacken

Spiel starten (unter ubuntu)

  1. Im Hauptmenü 'Spiele' dann 'squeeze' wählen.

Spiel starten (unter fedora)

  1. zum Ordner mit dem entpackten Archiv wechseln
  2. Spiel per Doppelklick auf 'squeeze' starten
  3. Auflösung und Vollbild-Modus können in der Startdatei eingestellt werden.

Spiel spielen

Spielziel

  1. Das Ziel in jedem Level ist es, den Bildschirm mit Kugeln zu füllen.
  2. Kugel innerhalb einer anderen Kugel zu erstellen ist verboten.

Hinweise

  1. Es sind 100 Punkte nötig um ins nächste Level zu kommen.
  2. Bei einem Crash wird dieses Limit erhöht.
  3. Große Kugeln geben mehr Punkte als kleine.
  4. ENTER drücken, wenn nichts mehr geht.
  5. Kollisionen während dem Aufpumpen von Kugeln sind OK. Beim Erstellen muss die Position aber legal sein.

Steuerung

  1. klicken&halten der linken Maustaste: Kugel aufpumpen.
  2. linke Maustaste loslassen: Kugel erstellen.
  3. enter: neues Spiel starten.
