Installation (under ubuntu)

  1. using ubuntu 8.04 (hardy) or earlier: run in terminal: 
    sudo apt-get install ruby rubygems libopengl-ruby libsdl-ruby1.8
  2. using ubuntu 8.10 (intrepid) or later: run in terminal: 
    sudo apt-get install ruby rubygems libopengl-ruby libsdl-ruby
  3. click here to download, select latest 'glgames'
  4. extract the archive

Installation (under fedora)

  1. run in root terminal: 
    yum groupinstall "Development Tools"
yum install ruby rubygems ruby-devel

yum install SDL-devel SDL_ttf-devel SDL_image-devel freeglut-devel
gem install rubysdl ruby-opengl

yum install bitstream-vera-fonts
  2. click here to download, select 'glgames'
  3. extract the archive

Starting the Game

  1. open the folder where you extracted the archive
  2. double-click 'tictactoe' to run the game

Target

  1. play TicTacToe
  2. get three in a row

Controls

  1. left mouse: set a piece
  2. right mouse: restart game
  3. middle mouse: let the computer move
